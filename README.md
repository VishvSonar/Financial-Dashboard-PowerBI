## Objective
Built an interactive financial dashboard to monitor company revenue, profitability, budget variance, cash flow, and receivables aging.

## Features
- KPI Cards (Revenue, Gross Margin %, EBITDA %, Net Cash)
- Monthly Revenue vs Profit Trend
- Budget vs Actual Variance Analysis
- Product/Service Performance
- Cash Flow Waterfall
- Receivables Aging Distribution
- Interactive Filters (Month, Region, Product)

## Tools Used
- Power BI
- DAX
- CSV/Excel
